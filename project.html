Day 1 — VS Code & GitHub Setup

Installation and setup of VS Code development environment.

VS Code Installation Steps

Step 1 — Download VS Code
Open your browser and visit the official Visual Studio Code website.
Click Download for Windows.

Step 2 — Run Installer
Open the downloaded .exe file.
Click Next → Accept Agreement → Next.

Step 3 — Select Important Options
During installation enable:
Add to PATH
Create Desktop Icon
Open with Code option

Step 4 — Finish Installation
Click Finish and launch VS Code successfully.

✅ Installation Complete!

Folder and File Creation Steps

Create a project folder.
Open VS Code → File → Open Folder.
Create HTML, CSS and JS files for development.

GitHub Setup

Step 1 — Create GitHub Account
Go to GitHub website and sign up using email ID.

Step 2 — Create New Repository
Click New Repository.
Enter repository name and click Create Repository.

Step 3 — Connect VS Code with GitHub
Initialize Git inside project folder.
Add files and commit changes.

Step 4 — Push Project Online
Link local repository with GitHub and push files using Git commands.

✅ GitHub Setup Completed!
